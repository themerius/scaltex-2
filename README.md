scaltex-2
=========

Install `bower` (http://bower.io/) for javascript dependency management.
Fetch the js dependencies with:

    cd public/scripts
    bower install

Install `sbt` (http://www.scala-sbt.org/), then simply run:

    cd ../..
    sbt run

View `http://localhost:8000/`
